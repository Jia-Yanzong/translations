Cadran riche en données avec 12 emplacements pour toutes vos données, affiché de manière simple. Entièrement personnalisable avec graphiques, barres de progression, 3 polices, etc.

⭐️ Gratuit vs Pro ⭐️
Ce cadran est disponible gratuitement avec toutes les fonctions de base. Les fonctionnalités premium (marquées avec [*]) nécessitent une Clé Pro pour débloquer. Vous pouvez acheter une Clé Pro ici: https://ko-fi.com/s/b509db80a0 Ou acheter la version PRO ici: https://apps.garmin.com/apps/e74a01b9-31e3-4798-871d-34eeeddc050d

⭐️ Configuration ⭐️
Note: Il y a deux endroits différents avec les paramètres:
* App Garmin IQ (sur votre téléphone): Configurez les champs de données, couleurs et paramètres généraux.
* Paramètres sur la montre: Accédez à des options plus détaillées comme le format de l'heure, les objectifs et les unités directement sur la montre en maintenant le bouton menu, sélectionnant le cadran et appuyant sur personnaliser.
