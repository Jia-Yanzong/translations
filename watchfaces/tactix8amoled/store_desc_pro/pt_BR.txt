⭐️ Configuração ⭐️
Nota: Existem dois lugares diferentes com configurações:
* App Garmin IQ (no seu telefone vá para o seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no relógio: Acesse opções mais detalhadas como formato de hora, metas e unidades diretamente no relógio segurando o botão menu, selecionando mostrador e tocando em personalizar.

⭐️ Campos de dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceirizados estão disponíveis, além de extras como suporte ao Monitor de Glicose (CGM), Open Weather Map (OWM) e suporte para outros aplicativos de terceiros.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields. Alguns campos de dados (como prontidão para treinamento ou HRV) atualmente não estão disponíveis para desenvolvedores terceirizados, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
