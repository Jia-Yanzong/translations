Mostrador limpo e elegante, inspirado no mostrador Marq Athlete.
Mais de 110 campos de dados, muitas opções AOD, personalizações ilimitadas, barras de progresso com gradiente e predefinições

⭐️ Grátis vs Pro ⭐️
Este mostrador está disponível sem custo com todas as funções básicas. Recursos premium (marcados com [*]) requerem uma Chave Pro para desbloquear. Você pode comprar uma Chave Pro aqui: https://watchface.io/marqathlete ou obter a versão PRO totalmente desbloqueada aqui: https://apps.garmin.com/apps/fb6d03ff-d520-4488-8c68-4c771006557d

⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador é totalmente otimizado para MIPS, mas as cores podem parecer diferentes como na maioria das imagens, pois estas são tiradas em um dispositivo Amoled. Você pode encontrar todos os temas integrados, incluindo temas MIPS aqui: https://watchface.io/marqathlete/themes

⭐️ Configuração ⭐️
Nota: Existem dois locais diferentes com configurações:
* App Garmin IQ (no seu telefone vá para seu dispositivo, mostradores, configurações): Configure campos de dados, cores e configurações gerais.
* Configurações no Relógio: Acesse opções mais detalhadas como formato de hora, objetivos e unidades diretamente no relógio segurando o botão menu, selecionando mostrador e tocando personalizar.

⭐️ Campos de Dados ⭐️
Todos os campos de dados que a Garmin atualmente fornece para desenvolvedores terceiros estão disponíveis, mais extras como suporte ao Monitor de Glicose (CGM), Open Weather Map (OWM) e suporte para outros apps terceiros.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields. Alguns campos de dados (como prontidão para treino ou HRV) atualmente não estão disponíveis para desenvolvedores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
