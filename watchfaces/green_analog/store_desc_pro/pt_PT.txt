Mostrador clássico, minimalista e altamente personalizável. Com mais de 100 pontos de dados, muitas opções AOD e baixo consumo de bateria, o mostrador perfeito para usar com estilo.


⭐️ Experimente antes de comprar! ⭐️
Antes de comprar este mostrador, verifique primeiro a versão gratuita para se certificar de que é o que pretende e que gosta de como fica no seu relógio: https://apps.garmin.com/apps/3ef6e4c3-8752-4563-980a-da183fc88a7c


⭐️ Dispositivos MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Este mostrador está totalmente optimizado para MIPS, mas as cores podem parecer diferentes como na maioria das imagens, pois estas foram tiradas num dispositivo Amoled. Pode encontrar todos os temas integrados aqui: https://watchface.io/green_analog/themes

⭐️ Configuração ⭐️
Nota: Há dois sítios diferentes com definições:
* App Garmin IQ (no seu telefone vá ao seu dispositivo, mostradores, definições): Configure campos de dados, cores e definições gerais.
* Definições no Relógio: Aceda a opções mais detalhadas como formato de hora, objectivos e unidades directamente no relógio mantendo premido o botão menu, seleccionando mostrador e tocando personalizar.

⭐️ Campos de Dados ⭐️
Todos os campos de dados que a Garmin actualmente fornece a programadores terceiros estão disponíveis, mais extras como suporte ao Monitor de Glicose (CGM), Open Weather Map (OWM) e suporte para outras apps terceiras.
Todos os campos de dados disponíveis estão listados aqui: https://watchface.io/docs/datafields. Alguns campos de dados (como prontidão para treino ou HRV) actualmente não estão disponíveis para programadores terceiros, veja aqui: https://watchface.io/faq/can-you-add-training-readiness
