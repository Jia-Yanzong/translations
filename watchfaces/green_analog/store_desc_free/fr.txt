Cadran classique, minimal et hautement personnalisable. Avec plus de 100 points de données, de nombreuses options AOD et une faible consommation de batterie, le cadran parfait à porter avec style.


⭐️ Gratuit vs Pro ⭐️
Ce cadran est disponible gratuitement avec toutes les fonctions de base. Les fonctionnalités premium (marquées avec [*]) nécessitent une Clé Pro pour être débloquées. Vous pouvez acheter une Clé Pro ici : https://apps.garmin.com/apps/87860c9d-4214-4b7c-9770-f2e0cf4ee78c

⭐️ Appareils MIPS (Fenix 7, Enduro, Forerunner 255, etc.) ⭐️
Ce cadran est entièrement optimisé pour MIPS, mais les couleurs peuvent paraître différentes comme dans la plupart des images, car celles-ci sont prises sur un appareil Amoled. Vous pouvez trouver tous les thèmes intégrés ici : https://watchface.io/green_analog/themes

⭐️ Configuration ⭐️
Note : Il y a deux endroits différents avec des paramètres :
* App Garmin IQ (sur votre téléphone allez à votre appareil, cadrans, paramètres) : Configurez les champs de données, les couleurs et les paramètres généraux.
* Paramètres sur la Montre : Accédez à des options plus détaillées comme le format de l'heure, les objectifs et les unités directement sur la montre en maintenant le bouton menu, en sélectionnant le cadran et en appuyant sur personnaliser.

⭐️ Champs de Données ⭐️
Tous les champs de données que Garmin fournit actuellement aux développeurs tiers sont disponibles, plus des extras comme le support du Moniteur de Glucose (CGM), Open Weather Map (OWM) et le support pour d'autres apps tierces.
Tous les champs de données disponibles sont listés ici : https://watchface.io/docs/datafields. Certains champs de données (comme la préparation à l'entraînement ou HRV) ne sont actuellement pas disponibles pour les développeurs tiers, voir ici : https://watchface.io/faq/can-you-add-training-readiness
